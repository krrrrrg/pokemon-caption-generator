# 포켓몬 2세대 스타일 자막 생성기

포켓몬 골드/실버 버전 스타일의 대화창을 만들어 PNG로 저장할 수 있는 웹 애플리케이션입니다.

## 기능

- 포켓몬 2세대 스타일의 픽셀 그래픽 대화창 생성
- 한글 픽셀 폰트 지원 (DungGeunMo)
- 실시간 미리보기
- PNG 이미지로 저장 (고해상도)

## 사용 방법

1. 텍스트 입력창에 원하는 문장을 입력합니다
2. 줄바꿈이 필요하면 엔터를 눌러주세요 (최대 2줄 권장)
3. "미리보기 업데이트" 버튼으로 결과를 확인합니다
4. "PNG로 저장하기" 버튼을 클릭하여 이미지를 다운로드합니다

## 기술 스택

- HTML5
- CSS3
- JavaScript
- html2canvas (이미지 캡처)
